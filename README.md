## Prerequisites

* MySQL (ver. 5.6.21 or newer): recommended XAMPP, https://www.apachefriends.org/index.html

* Libopf (ver. 2.1 or newer): http://www.ic.unicamp.br/~afalcao/libopf/
* Gnuplot: run `$ sudo apt-get install gnuplot`
